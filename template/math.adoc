=== 行内与独行

==== 行内公式

> 将公式插入到本行内，符号：`$公式内容$`

```
$f(x)=2x^2+3x+5$
```

_效果：_
$f(x)=2x^2+3x+5$

==== 独行公式

> 将公式插入到新的一行内，并且居中，符号：`$$公式内容$$`

```
$$x_{1,2} = {-b\pm\sqrt{b^2 - 4ac} \over 2a}.$$
```

_效果：_
$$x_{1,2} = {-b\pm\sqrt{b^2 - 4ac} \over 2a}.$$
